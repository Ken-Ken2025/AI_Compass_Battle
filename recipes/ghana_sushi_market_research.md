# ガーナ市場調査レポート：「未来のお寿司」販売戦略

## 1. ターゲット市場の概要

**主要商圏：アクラ（首都）**

- 全国消費支出の 32% を占める最大消費市場
- 国際チェーン（バーガーキング・KFC・ピザハット）が相次いで進出中
- 若年層・都市部富裕層を中心に「外食＝ステータス」の意識が定着

**日本食の先行事例**

- **Santoku（アクラ）**：西アフリカ初のハイエンド日本食レストラン。ロンドンNOBU出身のシェフが2012年創業。現地における日本食のベンチマーク。
- 中東・アフリカ地域の寿司市場シェアは世界の 12%。年間7.3%成長中（2025→2026）

---

## 2. ガーナ人の食の特性

| 項目 | 内容 |
|------|------|
| 主食 | フフ（芋類をつく料理）、ジョロフライス、ワケイェ |
| 人気食材 | キャッサバ、プランテン、トマト、ピーナッツ、パームオイル、オクラ |
| 人気の魚 | ティラピア（最重要）、ナマズ、イワシ、カニ、バラクーダ |
| 調味料 | 生姜、唐辛子（シト）、アリゲーターペッパー、カラバシュナツメグ |
| 口の傾向 | スパイシー・旨み強め・ボリューム重視 |

---

## 3. 宗教・食の制約

- ガーナ人口の約 20〜30% がムスリム（北部に集中、都市部にも在住）
- **ハラール需要が急拡大中** → 食品企業がハラール認証取得を戦略化
- スケールのある魚（ティラピア・サーモン等）はハラール対応
- 豚・アルコール（みりん・日本酒）は除外が必要
- **ハラール認証取得で市場を一気に広げられる**

---

## 4. 課題・リスク

| 課題 | 内容 |
|------|------|
| 価格感度 | インフレで家計が圧迫。高単価商品は富裕層・観光客向け限定 |
| 生魚への抵抗 | ガーナ料理に生食文化はほぼない。フライ・グリルが主流 |
| 冷蔵チェーン | 鮮度管理インフラの整備状況を要確認 |
| 認知度 | 寿司の概念自体を説明するマーケティングが必要 |

---

## 5. 推奨ターゲット層とチャネル

| 層 | チャネル |
|----|----------|
| 富裕層・在住外国人 | アクラ高級ショッピングモール内レストラン |
| 若年都市層（25〜35歳） | フードデリバリー（Bolt Food等）× SNS |
| 観光客 | ホテル内デリカウンター |

---

## 6. 総括

ガーナには日本食の素地（Santokuが開拓済み）があり、若い都市層の新しい食への好奇心は高まっている。ただし生魚への抵抗感・価格感度・ハラール対応が三大ハードル。

**「焼き・揚げ・地元食材融合」×「ハラール認証」×「映える見た目」** の三本柱で攻める。
