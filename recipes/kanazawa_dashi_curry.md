# 金沢出汁カレー

**カテゴリ**: 和風カレー / メインディッシュ  
**調理時間**: 約60分  
**分量**: 4人前

---

## 材料

### 出汁（だし）
| 材料 | 分量 |
|------|------|
| 昆布（利尻または羅臼） | 15g |
| 鰹節 | 20g |
| 水 | 1200ml |

### 具材
| 材料 | 分量 |
|------|------|
| 鶏もも肉 | 400g |
| 加賀れんこん | 200g |
| 金時草（または三つ葉） | 50g |
| 玉ねぎ | 2個 |
| にんじん | 1本 |
| じゃがいも | 2個 |

### カレーベース
| 材料 | 分量 |
|------|------|
| カレールウ | 4〜5片 |
| 酒 | 大さじ2 |
| みりん | 大さじ1 |
| 薄口醤油 | 大さじ1 |
| 生姜（すりおろし） | 1片分 |
| サラダ油 | 大さじ1 |

---

## 調理手順

**1. 出汁を引く**  
昆布を水に30分浸けてから中火にかけ、沸騰直前に取り出す。鰹節を加えて2〜3分煮出し、濾す。

**2. 野菜の下処理**  
玉ねぎはくし切り、にんじん・じゃがいもは乱切り、加賀れんこんは1cm厚の輪切りにして酢水にさらす。

**3. 鶏肉を炒める**  
鍋に油を熱し、鶏もも肉を皮目から焼いて表面に焼き色をつける。取り出して一口大に切る。

**4. 野菜を炒める**  
同じ鍋で玉ねぎを狐色になるまで炒め、にんじん・れんこんを加えてさらに2〜3分炒める。

**5. 出汁で煮込む**  
引いた出汁・酒・みりんを加え、鶏肉を戻す。じゃがいもも加えて蓋をして15分ほど煮込む。

**6. ルウを加える**  
火を弱め、カレールウを溶かし入れる。薄口醤油・生姜を加えて弱火で10分ほど煮る。

**7. 仕上げ**  
金時草（または三つ葉）をさっと加えてひと煮立ちさせ、火を止める。

---

## ポイント・コツ

- **昆布は必ず水から**：煮立てると粘りが出るので沸騰前に引き上げる
- **薄口醤油で上品に**：普通の醤油より色が淡く、出汁の風味が際立つ
- **れんこんの食感を活かす**：煮すぎると崩れるので後半に投入してもOK
- **出汁が命**：市販の昆布だしパックでも代用可だが、本出汁で格段に違いが出る

---

## アレンジ提案

- **加賀野菜バリエーション**: 五郎島金時（さつまいも）や加賀太きゅうりを使っても美味
- **魚介バリエーション**: 鶏肉をカニや白身魚に替えると金沢らしさがさらに増す
- **ルウアレンジ**: 白みそを少量（大さじ1）加えるとコクが増す
