## 基本：キャッサバ寿司飯

*全レシピ共通の土台。地産地消アピール用*

### 材料（約4人分）

| 材料 | 分量 |
|------|------|
| 日本米 | 300g |
| キャッサバ粉（カサバフラワー） | 50g |
| 米酢 | 50ml |
| 砂糖 | 20g |
| 塩 | 8g |

### 手順

1. 米とキャッサバ粉を合わせて水360mlで炊く
2. 酢・砂糖・塩を合わせて電子レンジで15秒温め、砂糖と塩を溶かす
3. 炊き上がったご飯に合わせ酢をしゃもじで切るように混ぜ、うちわで扇いで冷ます

> キャッサバ粉10〜15%混合で、ガーナ人になじみのあるもちっとした食感に近づく。

## レシピ①：ティラピア焼き握り

*「ガーナの国民魚×寿司」最もなじみやすい入口商品*

### 材料（2人分・8貫）

| 材料 | 分量 |
|------|------|
| ティラピア切り身 | 200g |
| キャッサバ寿司飯 | 280g |
| 唐辛子（フレッシュ赤） | 2本 |
| 干しエビ | 大さじ1 |
| にんにく | 1片 |
| パームオイル | 大さじ2 |
| 生姜（スライス） | 適量 |
| ライム | 1/2個 |
| 塩 | 少量 |

### 手順

1. **シトソース**：唐辛子・干しエビ・にんにくをフードプロセッサーで粗めにペースト状にし、パームオイルで5分炒めて冷ます
2. ティラピアに塩・生姜をまぶし、スキレットまたは炭火で両面を香ばしくグリル（中心温度70℃以上）
3. 皮目をカリッと仕上げ、一口大（4cm）にカット
4. 寿司飯を握り（俵形）、上にティラピアをのせる
5. シトソースを少量のせ、ライムを搾ってサーブ

## レシピ②：プランテン＆ピーナッツロール

*揚げプランテン×ピーナッツバター×酢飯の完全ローカルロール*

### 材料（2本分・12切れ）

| 材料 | 分量 |
|------|------|
| キャッサバ寿司飯 | 320g |
| 焼き海苔 | 2枚 |
| プランテン（熟れ気味・黄色） | 1本 |
| ピーナッツバター（無糖・粒あり） | 大さじ3 |
| ハチミツ | 小さじ1 |
| 揚げ油 | 適量 |
| 唐辛子パウダー（カイエン） | 少量 |
| 塩 | 少量 |

### 手順

1. プランテンを縦に薄切り（5mm）にし、170℃の油で3〜4分揚げて塩をふる
2. ピーナッツバターにハチミツと唐辛子パウダーを混ぜてソースを作る
3. 海苔の上に寿司飯を薄く広げ、手前にプランテンを並べピーナッツソースを塗る
4. 手前からしっかり巻いて、ラップで形を整え1分おく
5. ぬれ包丁で6等分にカット

## レシピ③：ピーナッツ味噌軍艦

*ガーナ産ピーナッツを「アフリカの味噌」として使う発酵フュージョン*

### 材料（8個分）

| 材料 | 分量 |
|------|------|
| キャッサバ寿司飯 | 240g |
| 焼き海苔（帯切り） | 8枚 |
| ピーナッツバター（無糖） | 大さじ4 |
| 白味噌 | 大さじ1 |
| ライムジュース | 小さじ2 |
| 生姜すりおろし | 小さじ1 |
| はちみつ | 小さじ1 |
| ごま油 | 小さじ1 |
| キュウリ（角切り） | 40g |
| 炒りごま | 適量 |

### 手順

1. ピーナッツ味噌の材料をすべて混ぜ合わせ、なめらかになるまで練る
2. 寿司飯を小判形に握り、帯状の海苔を巻いて軍艦の形に整える
3. 上にピーナッツ味噌を山盛りのせ、キュウリの角切りと炒りごまを飾る

> 白味噌を少量だけ使うことで発酵の深みが加わり、ピーナッツバターだけより格段にリッチな味になる。

## レシピ④：昆虫プロテイン・クリスピー軍艦

*「未来食材」フライドクリケットで2050年の寿司を体現*

### 材料（8個分）

| 材料 | 分量 |
|------|------|
| キャッサバ寿司飯 | 240g |
| 焼き海苔（帯切り） | 8枚 |
| フライドクリケット（市販・塩味） | 30g |
| アボカド | 1個 |
| ライムジュース | 小さじ2 |
| 塩・黒コショウ | 少量 |
| ライムゼスト | 少量 |

### 手順

1. アボカドをフォークで潰し、ライムジュース・塩・黒コショウで味を整えてワカモレ状にする
2. 寿司飯を握り、海苔で軍艦にする
3. アボカドクリームをのせ、フライドクリケットを5〜6匹トッピング
4. ライムゼストをひとつまみ散らす

> アフリカでは昆虫食の受容度が高い（特に西アフリカ）。「プロテイン豊富な未来食」としてZ世代・健康意識層にアピール。

## サイド：パームオイル赤だし風スープ

*寿司に合わせる「ガーナ版味噌汁」で体験を完結させる*

### 材料（4人分）

| 材料 | 分量 |
|------|------|
| 出汁（煮干し）または野菜ブロス | 600ml |
| パームオイル | 大さじ1 |
| オクラ（輪切り） | 6本分 |
| トマト（ざく切り） | 1個 |
| 玉ねぎ（みじん切り） | 1/4個 |
| 干しエビ | 大さじ1 |
| 塩・唐辛子 | 適量 |

### 手順

1. パームオイルで玉ねぎを炒め、トマトを加えて5分煮詰める
2. ブロスと干しエビを加えて沸騰させる
3. オクラを加えて3分煮る（とろみが出る）
4. 塩・唐辛子で味を整えてサーブ
