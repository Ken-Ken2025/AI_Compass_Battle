# 名古屋の味噌カレー

**カテゴリ**: 和風カレー / メインディッシュ
**調理時間**: 約75分
**分量**: 4人前

---

## 材料

### 味噌ベース
| 材料 | 分量 |
|------|------|
| 八丁味噌 | 80g |
| 信州赤味噌（補助用） | 30g |
| みりん | 大さじ2 |
| 酒 | 大さじ2 |
| 砂糖（ザラメ可） | 大さじ1 |

### 出汁
| 材料 | 分量 |
|------|------|
| 鰹節 | 15g |
| 昆布 | 10g |
| 水 | 1000ml |

### 具材
| 材料 | 分量 |
|------|------|
| 鶏もも肉（名古屋コーチン推奨） | 500g |
| 玉ねぎ | 2個 |
| にんじん | 1本 |
| じゃがいも（メークイン） | 2個 |
| しめじ | 1パック |
| にんにく（すりおろし） | 1片 |
| 生姜（すりおろし） | 1片 |

### カレーベース
| 材料 | 分量 |
|------|------|
| カレールウ（中辛） | 3〜4片 |
| サラダ油 | 大さじ1 |
| バター | 10g |
| 山椒 | 少々 |

## 調理手順

**1. 出汁を引く**
昆布を水に30分浸けてから中火にかけ、沸騰直前で取り出す。鰹節を加えて2〜3分煮出し、濾して出汁を取る。

**2. 味噌ダレを合わせる**
八丁味噌・赤味噌・みりん・酒・砂糖を小鍋に入れ、弱火で練りながら滑らかになるまで混ぜておく。

**3. 野菜の下処理**
玉ねぎは粗みじん切り、にんじん・じゃがいもは乱切り、しめじは石づきを取って小房に分ける。

**4. 鶏肉を焼く**
鍋にサラダ油を熱し、塩こしょうした鶏もも肉を皮目から焼いて両面に焼き色をつける。取り出して一口大に切る。

**5. 玉ねぎを飴色に炒める**
同じ鍋で玉ねぎを中弱火でじっくり炒め、飴色になるまで15分ほどかける。にんにく・生姜を加えて香りを立てる。

**6. 出汁で煮込む**
鶏肉を戻し、にんじん・じゃがいも・しめじを加えて軽く炒める。出汁を注いで沸騰させ、アクを取りながら15分煮込む。

**7. 味噌とルウを溶く**
火を弱め、合わせておいた味噌ダレを煮汁で少しずつ溶きながら加える。続けてカレールウを溶かし入れ、10分弱火で煮る。

**8. 仕上げ**
バターを加えて全体をなじませ、火を止める直前に山椒をひと振り。

---

## ポイント・コツ

- **八丁味噌は塩気と渋みが強い**：必ず砂糖・みりんで甘みを補い、ルウは少なめにして味噌の風味を主役にする
- **二種味噌でバランスを取る**：八丁単体だと重いので、まろやかな赤味噌を3:1で合わせるとコクと飲み口が両立する
- **玉ねぎは飴色必須**：甘みが味噌の渋みを和らげる土台になる
- **出汁は鰹昆布の合わせが鉄則**：洋風ブイヨンより和出汁が八丁味噌と相性抜群
- **追い山椒で名古屋らしさ**：器に盛ってから山椒をもうひと振りすると味が締まる

## アレンジ提案

- **味噌煮込みカレーうどん**: 残ったカレーにきしめん（または味噌煮込み用うどん）を入れて土鍋でひと煮立ち
- **名古屋風カレーおじや**: ごはん・溶き卵を加えて朝食用に。ネギと刻み海苔を添える
- **手羽先トッピング**: 別焼きした名古屋風手羽先を上に乗せて「ザ・名古屋」仕様に
- **赤味噌マシマシ**: 八丁味噌を100gまで増やすと、より濃厚で大人向けの仕上がり
