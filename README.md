# AI Compass Dev Battle — 料理レシピリポジトリ

## このリポジトリの目的

**AI Compass Dev Battle** は、複数メンバーで Git / GitHub を使った共同開発を練習するためのプロジェクトです。

テーマを「料理レシピ」に設定し、以下のような開発の基本操作を体験・習得することを目的としています。

- ブランチの作成・切り替え
- コミット・プッシュ・プル
- プルリクエスト（PR）の作成とレビュー
- コンフリクトの解消
- チームでのコードレビュー文化の醸成

## ディレクトリ構成

```
AI_Compass_Dev_Battle/
├── README.md          ← このファイル
└── recipes/           ← レシピファイル置き場
    └── ramen.md       ← サンプル：醤油ラーメン
```

## 参加ルール

1. 作業は必ず **自分のブランチ** を切ってから行う（例: `feature/your-name-recipe`）
2. `main` ブランチへの直接プッシュは禁止
3. 作業完了後はプルリクエストを作成し、メンバーのレビューを受けてからマージする
4. コミットメッセージは日本語でも英語でも OK。内容が分かるように書く

## レシピの追加方法

1. `recipes/` フォルダに `your-recipe-name.md` を作成する
2. テンプレート（後述）に従って書く
3. PR を出す

## レシピテンプレート

```markdown
# レシピ名

## 材料（〇人分）
- 材料A: 量
- 材料B: 量

## 手順
1. 手順1
2. 手順2

## メモ
- コツや注意点など
```
